Клонирование ппроэкта в папку nameeProject:
git clone https://... nameProject

Добавить файл в коммит:
git add .

Записать коммит:
git commit -m "Комментарий коммита"

Отправить коммит:
git push

Принять изменения:
git pull

Авторство коммитов:
git config --global user.email "Ваш емаил"
git config --global user.name "Ваше имя"
